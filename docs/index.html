KDE.CSS

CSS Library, that aims to provide ability of looking like real KDE apps to your HTML pages.

Currently these things are supported:

  • Buttons (primary, secondary, inline)
  • Text and file inputs, textarea
  • Combobox (select element)
  • Checkboxes and radio buttons
  • Slider (partial)
  • Fieldset (partial)
  • Progressbar
  • Images
  • Tabs
  • Windows (partial and useless)

A window is a container, that can have header, menu, tabs and body.
Window consists of header section and main section: 

<div class="kde-window" style="width: 600px; height: 480px;">
    <header>
        Window title
    </header>
    <main>
        <nav>
            <a class="kde-menu-link" href="#!File">File</a>
        </nav>
        <div class="tabs">
            <div class="tab active">       <!-- only one tab can be active at one time -->
                <span class="aux">*</span> <!-- located to the left of the tab -->
                main.c
                <span class="close">×</span>
            </div>
            <div class="tab">
                oto.ini
                <span class="close">×</span>
            </div>
        </div>
        
        Window content
    </main>
</div>
